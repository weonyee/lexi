# Lexi - 드래그 번역

웹페이지에서 모르는 단어를 드래그하면 바로 한국어 번역이 나타납니다. 탭 전환 없이, 복사 붙여넣기 없이.

---

## 설치 방법

1. 크롬 웹스토어에서 **Lexi** 를 검색하거나 아래 링크로 이동합니다.
2. **Chrome에 추가** 버튼을 클릭합니다.
3. 팝업에서 **확장 프로그램 추가** 를 누르면 설치 완료입니다.

설치 후 크롬 툴바에 Lexi 아이콘이 나타납니다.

---

## 사용 방법

1. 번역하고 싶은 외국어 단어나 구문을 마우스로 **드래그** 합니다.
2. 드래그한 텍스트 바로 위에 **한국어 번역 팝업** 이 나타납니다.
3. 확인 후 팝업 바깥을 클릭하거나 `ESC` 키를 누르면 닫힙니다.

---

## 이런 경우 팝업이 뜨지 않아요

- 한국어 텍스트를 드래그한 경우
- 숫자나 기호만 드래그한 경우
- 3단어 이상 드래그한 경우

---

## 팁

- 단어 하나만 드래그하면 번역과 함께 **품사 및 여러 의미** 를 사전처럼 볼 수 있습니다.
- 팝업 위에 마우스를 올려두면 **자동 닫힘이 일시 중지** 됩니다.
- 영어, 일본어, 중국어 등 언어를 따로 설정하지 않아도 **자동 감지** 됩니다.
