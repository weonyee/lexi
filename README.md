# 📚 Quick Word Translator

웹페이지에서 단어를 드래그하면 자동으로 번역과 사전 정의를 보여주는 크롬 익스텐션입니다.

## ✨ 기능

- **드래그 번역**: 웹페이지에서 모르는 단어나 문장을 드래그하면 자동으로 번역 팝업이 나타납니다
- **다국어 지원**: 한국어, 영어, 일본어, 중국어 등 13개 언어로 번역 가능
- **사전 정의**: 단어의 품사와 다양한 뜻을 함께 보여줍니다
- **언어 자동 감지**: 원본 텍스트의 언어를 자동으로 감지합니다

## 🚀 설치 방법

1. **크롬 브라우저**에서 `chrome://extensions/` 주소로 이동합니다
2. 우측 상단의 **개발자 모드**를 활성화합니다
3. **압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭합니다
4. `translate` 폴더를 선택합니다
5. 설치 완료! 🎉

## 📖 사용 방법

1. 익스텐션 아이콘을 클릭하여 번역할 언어를 선택합니다 (기본값: 한국어)
2. 웹페이지에서 번역하고 싶은 단어나 문장을 **드래그**합니다
3. 자동으로 번역 팝업이 나타납니다
4. 팝업 외부를 클릭하거나 ESC 키를 누르면 팝업이 닫힙니다
