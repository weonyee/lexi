# Lexi - 드래그 번역 익스텐션

웹 서핑 중 모르는 단어가 나올 때마다 새 탭을 열어 검색하고 계신가요? Lexi는 그 번거로움을 없애줍니다. 드래그 한 번으로 한국어 번역 결과가 바로 눈앞에 나타납니다.

---

## 왜 Lexi를 설치해야 할까요?

- 흐름이 끊기지 않습니다 — 페이지를 벗어나지 않고 그 자리에서 바로 번역 결과를 확인합니다.
- 단어 뜻까지 함께 — 단순 번역을 넘어 품사와 다양한 의미를 사전처럼 보여줍니다.
- 언어를 자동으로 감지합니다 — 영어든 일본어든 중국어든, 원본 언어를 따로 설정할 필요가 없습니다.
- 모든 번역은 한국어로 — 어떤 언어를 드래그해도 항상 한국어로 번역됩니다.
- 다크모드 지원 — OS/크롬 다크모드에서도 팝업이 자연스럽게 어두운 테마로 표시됩니다.
- 가볍고 빠릅니다 — 별도 서버 없이 Google Translate API를 직접 호출합니다.

---

## 주요 기능

**드래그 번역**
웹페이지에서 단어나 짧은 구문(2단어 이하)을 드래그하면 선택 위치 근처에 번역 팝업이 자동으로 나타납니다.

**사전 정의 표시**
단어 단위로 드래그하면 번역과 함께 품사(명사, 동사 등)와 여러 의미를 함께 보여줍니다.

**팝업 자동 닫힘**
팝업 외부를 클릭하거나 ESC 키를 누르면 닫힙니다. 마우스를 팝업 위에 올려두면 3초 자동 닫힘이 일시 중지됩니다.

**다크모드 대응**
`prefers-color-scheme: dark` 기준으로 팝업 색상이 자동 전환됩니다.

## 설치 방법

1. 이 저장소를 다운로드하거나 클론합니다.
   ```
   git clone https://github.com/your-repo/lexi.git
   ```
   또는 GitHub에서 `Code → Download ZIP` 후 압축을 해제합니다.

2. 크롬 브라우저 주소창에 `chrome://extensions/` 를 입력해 확장 프로그램 관리 페이지로 이동합니다.

3. 우측 상단의 **개발자 모드** 토글을 켭니다.

4. **압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭합니다.

5. 다운로드한 프로젝트 폴더를 선택합니다.

6. 설치 완료. 크롬 툴바에 Lexi 아이콘이 나타납니다.

> 파일 수정 후에는 `chrome://extensions/`에서 Lexi 카드의 새로고침(↺) 버튼을 누른 뒤, 해당 탭을 닫았다가 다시 열어야 반영됩니다.

## 사용 방법

**기본 사용**

1. 번역하고 싶은 외국어 단어나 구문을 마우스로 드래그합니다.
2. 드래그한 텍스트 위쪽에 한국어 번역 팝업이 자동으로 나타납니다.
3. 팝업 확인 후 외부 클릭 또는 `ESC` 키로 닫습니다. 3초 후 자동으로도 닫힙니다.

**팝업이 뜨지 않는 경우**

아래 경우에는 의도적으로 팝업이 표시되지 않습니다.

- 한국어 텍스트를 드래그한 경우 (번역 대상이 아님)
- 숫자, 기호만 드래그한 경우
- 3단어 이상 드래그한 경우

**사전 정의 보기**

단어 하나만 드래그하면 번역 결과와 함께 품사(명사, 동사 등)와 여러 의미를 사전처럼 확인할 수 있습니다.

**팝업 유지하기**

팝업 위에 마우스를 올려두면 3초 자동 닫힘이 일시 중지됩니다.
